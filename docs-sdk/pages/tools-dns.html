DNS & Domains Tools

8 tools for managing .ton domains: status checks, auction participation, wallet linking, and TON Site configuration. Domain names accept both example and example.ton formats.

Lookup

dns_check

Check a .ton domain status: available, in auction, or owned. Returns owner, expiry, current bid, and estimated minimum price.

ParameterTypeDescription
domainstringDomain name (4–126 chars, with or without .ton)

dns_resolve

Resolve a .ton domain to its linked wallet address.

ParameterTypeDescription
domainstringDomain to resolve (e.g. alice.ton)

dns_auctions

List active .ton domain auctions with current bids and end times.

ParameterTypeDescription
tldstring?Top-level domain filter (default: ton)

Auctions

dns_start_auction

Start an auction for an unminted .ton domain by placing the opening bid.

ParameterTypeDescription
domainstringDomain name to auction (with or without .ton)
amountnumberOpening bid in TON (min depends on domain length)

dns_bid

Place a bid on an active .ton domain auction. Bid must be ≥ 105% of the current bid.

ParameterTypeDescription
domainstringDomain name (with or without .ton)
amountnumberBid amount in TON (must be ≥ 105% of current bid, min: 1)

Records

dns_link

Link your wallet address to a .ton domain you own.

ParameterTypeDescription
domainstringDomain you own (with or without .ton)
addressstring?Wallet address to link (defaults to your wallet)

dns_unlink

Remove the wallet address linked to a .ton domain.

ParameterTypeDescription
domainstringDomain to unlink (with or without .ton)

dns_set_site

Set a TON Site (ADNL) record on a domain, pointing it to a TON Web server.

ParameterTypeDescription
domainstringDomain you own (with or without .ton)
adnlstringADNL address of your TON Site
