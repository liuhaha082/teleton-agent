Telegram Tools

76 tools covering messaging, chats, contacts, groups, media, interactive features, folders, profile, stickers, Stars, and gifts. Tools marked user require userbot mode; bot requires bot mode.

Messaging (13)

telegram_send_message

Send a text message to a Telegram chat. For custom keyboards use telegram_reply_keyboard; for media use telegram_send_photo/gif/sticker.

ParameterTypeDescription
chatIdstringChat ID to send to (e.g. -1001234567890, @username)
textstringMessage text (max 4096 characters)
replyToIdnumber?Message ID to reply to

telegram_edit_message

Modify a previously sent message in-place. Only your own messages, within 48h. Supports Markdown formatting.

ParameterTypeDescription
chatIdstringChat ID where the message was sent
messageIdnumberID of the message to edit
textstringNew text content (max 4096 characters)

telegram_delete_message

Delete messages from a chat. Own messages in any chat, or any message with admin rights. Deletion is permanent.

ParameterTypeDescription
chatIdstringChat ID where the messages are located
messageIdsnumber[]Array of message IDs to delete
revokeboolean?Delete for everyone (default: true)

telegram_forward_message

Forward one or more messages from one chat to another. Shows original sender attribution.

ParameterTypeDescription
fromChatIdstringSource chat ID
toChatIdstringDestination chat ID
messageIdsnumber[]Message IDs to forward
silentboolean?Send without notification (default: false)
backgroundboolean?Forward in background (default: false)

telegram_quote_reply user

Reply to a specific excerpt within a message, quoting a portion of the original text.

ParameterTypeDescription
chatIdstringChat ID
messageIdnumberMessage to reply to
textstringReply text
quotestring?Excerpt to quote from the original message

telegram_get_replies user

Fetch all replies in a message thread.

ParameterTypeDescription
chatIdstringChat ID containing the thread
messageIdnumberRoot message ID of the thread
limitnumber?Max replies to return (default: 50)

telegram_search_messages user

Search messages by text query in a chat or globally.

ParameterTypeDescription
querystringSearch query string
chatIdstring?Limit search to this chat (omit for global)
limitnumber?Max results (default: 20)

telegram_schedule_message user

Queue a message for delayed delivery at a specified time.

ParameterTypeDescription
chatIdstringChat ID
textstringMessage text

telegram_get_scheduled_messages user

List pending scheduled messages in a chat.

ParameterTypeDescription
chatIdstringChat ID to list scheduled messages for

telegram_delete_scheduled_message user

Cancel a scheduled message before it is sent.

ParameterTypeDescription
chatIdstringChat ID
messageIdnumberScheduled message ID to cancel

telegram_send_scheduled_now user

Send a scheduled message immediately instead of waiting for its delivery time.

ParameterTypeDescription
chatIdstringChat ID
messageIdnumberScheduled message ID to send now

telegram_pin_message

Pin a message in a chat. Requires admin rights in groups/channels.

ParameterTypeDescription
chat_idstringChat ID or username
message_idnumberID of the message to pin
silentboolean?Pin without notification (default: false)
both_sidesboolean?Pin for both sides in private chats (default: true)

telegram_unpin_message user

Unpin a message or all messages in a chat. Requires admin rights in groups/channels.

ParameterTypeDescription
chat_idstringChat ID or username
message_idnumber?Message ID to unpin (omit to unpin latest)
unpin_allboolean?Unpin all pinned messages (default: false)
