DEX Trading Tools

10 tools for trading on TON DEXes. Use stonfi_quote / dedust_quote to preview rates, then stonfi_swap / dedust_swap to execute. For cross-DEX comparison, use dex_quote on the TON tools page.

STON.fi (5)

stonfi_quote

Get a swap price quote from STON.fi. Preview only, does not execute.

ParameterTypeDescription
from_assetstrington or jetton master address (EQ… format)
to_assetstrington or jetton master address (EQ… format)
amountnumberAmount in human-readable units (e.g. 10 for 10 TON)
slippagenumber?Slippage tolerance (0.01 = 1%, default: 0.01)

stonfi_swap

Execute a token swap on STON.fi. Supports TON↔jetton and jetton↔jetton. Use stonfi_quote first to preview.

ParameterTypeDescription
from_assetstrington or jetton master address. Always pass ton as a string.
to_assetstrington or jetton master address. Always pass ton as a string.
amountnumberAmount in human-readable units (min: 0.001)
slippagenumber?Slippage tolerance (0.01 = 1%, default: 0.01, max: 0.5)

stonfi_search

Search jettons by name or symbol on STON.fi.

ParameterTypeDescription
querystringToken name or ticker symbol to search
limitnumber?Max results (default: 10)

stonfi_trending

Get trending jettons on STON.fi by volume.

ParameterTypeDescription
limitnumber?Max results (default: 20)

stonfi_pools

List active STON.fi liquidity pools with TVL and volume.

ParameterTypeDescription
limitnumber?Max pools (default: 20)

DeDust (5)

dedust_quote

Get a swap price quote from DeDust. Preview only, does not execute.

ParameterTypeDescription
from_assetstrington or jetton master address (EQ… format)
to_assetstrington or jetton master address (EQ… format)
amountnumberAmount in human-readable units (min: 0.001)
pool_typestring?volatile (default) or stable
slippagenumber?Slippage tolerance (0.01 = 1%, default: 0.01)

dedust_swap

Execute a token swap on DeDust. Supports TON↔jetton and jetton↔jetton. Use dedust_quote first to preview.

ParameterTypeDescription
from_assetstrington or jetton master address. Always pass ton as a string.
to_assetstrington or jetton master address. Always pass ton as a string.
amountnumberAmount in human-readable units (min: 0.001)
pool_typestring?volatile (default) or stable for stablecoin pairs
slippagenumber?Slippage tolerance (0.01 = 1%, default: 0.01, max: 0.5)

dedust_pools

List DeDust liquidity pools.

ParameterTypeDescription
No parameters.

dedust_prices

Get real-time token prices from DeDust.

ParameterTypeDescription
assetsstring[]?Jetton addresses to fetch prices for (omit for all)

dedust_token_info

Get token info from DeDust by address or symbol.

ParameterTypeDescription
addressstringJetton master address or token symbol
